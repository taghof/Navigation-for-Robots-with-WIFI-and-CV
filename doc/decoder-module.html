Module decoder

source code

Video decoding for the AR.Drone.

This library uses psyco to speed-up the decoding process. It is however written in a way that it works also without psyco installed. On the author's development machine the speed up is from 2FPS w/o psyco to > 20 FPS w/ psyco.
